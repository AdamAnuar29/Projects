Welcome to the GEVS (General Election Voting System) Open Data Rest API

REST API available

  1. to obtain the electrols district's vote count
    • API Format: GET /gevs/constituency/northern-kunlun-mountain
    • http://[server]:[port]/gevs/constituency/northern-kunlun-mountain
    • Example: http://localhost:3000/gevs/constituency/northern-kunlun-mountain

  2. to obtain all the results
    • API Format: GET /gevs/results
    • Example: http://localhost:3000/gevs/results
